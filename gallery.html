Arquitecturas en la Programación Paralela

Memoria Compartida

Múltiples procesadores comparten un espacio de memoria común. Cada procesador puede acceder y modificar los datos almacenados en esta memoria compartida.

Ver más

Arquitectura de Memoria Distribuida

Múltiples computadoras independientes están conectadas mediante una red de comunicación. Cada computadora tiene su propia memoria local y no comparte directamente su memoria con las demás.

Ver más

Arquitectura Híbrida

Combina elementos de las arquitecturas de memoria compartida y memoria distribuida. Los sistemas híbridos pueden incluir múltiples nodos con memoria distribuida, donde cada nodo tiene su propia memoria local y comparte parte de ella con otros nodos mediante una memoria compartida.

Ver más

Arquitectura de Clúster

Múltiples computadoras independientes, también conocidas como nodos, se interconectan mediante una red de alta velocidad para formar un clúster de cómputo. Cada nodo puede tener su propia arquitectura de memoria (compartida o distribuida).
